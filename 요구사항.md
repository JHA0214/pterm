# PTerm
물리치료 관련 종사자와 학생을 위한 의학·물리치료 용어 검색 웹페이지
비상업 프로젝트이며, 수익화나 유료화를 목표로 하지 않는다.

## 기능
검색기능을 통해 용어(한글명, 영문명, 약어)를 입력해 원하는 용어를 찾을 수 있다.
검색어는 용어명뿐 아니라 관련 키워드로도 찾을 수 있다.
검색 단락 밑에 분류 버튼으로 카테고리를 고를 수 있게 한다(예: 해부학, 운동학, 평가·측정, 치료기법 등). 분류 선택 시 하위 세부 항목을 고를 수 있는 페이지가 나온다.
각 용어는 상세 페이지에서 정의, 관련 용어, 출처(참고 문헌 또는 링크)를 함께 보여준다.

## 콘텐츠 작성 원칙
용어 설명은 특정 교과서·사전의 문장을 그대로 복사하지 않고 직접 풀어 쓰거나, 공공누리·CC 라이선스 등 재배포가 허용된 자료를 출처 표시와 함께 사용한다.
저작권이 불분명한 자료는 데이터로 포함하지 않으며, 필요한 경우 원문 대신 출처 링크만 제공한다.

## 면책 문구
모든 페이지 또는 하단에 "본 서비스는 학습·참고 목적이며 실제 임상 판단을 대체하지 않는다"는 안내 문구를 표시한다.

## 저장 방식 (제안)
초기 버전은 별도 서버·데이터베이스 없이 정적 데이터(JSON 등)로 용어를 관리하는 방향을 우선 고려한다.
추후 용어 수가 많아지거나 사용자 기여(용어 제안 등) 기능이 필요해지면 서버 도입을 검토한다.

## 배포
비상업 개인/학습 프로젝트로 GitHub 등에 공개할 수 있으며, 공개 저장소일 경우 포함되는 모든 데이터 파일도 공개된다는 점을 감안하여 콘텐츠 원칙을 지킨다.
